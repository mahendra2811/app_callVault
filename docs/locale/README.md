# docs/locale — onboarding for React / Next.js developers

This folder is for you if you've shipped React + Next.js apps and now have to work in CallVault (a native Android app written in Kotlin + Jetpack Compose). It translates the unfamiliar parts back into things you already know.

Read in order:

| # | File | What it covers |
|---|------|----------------|
| 1 | `00-react-dev-orientation.md` | Mental model: there is no dev server. Here's the equivalent loop, mapped term-by-term. |
| 2 | `01-setup.md` | Install JDK, Android Studio, the Android SDK, and connect a phone. One-time setup. |
| 3 | `02-run-the-app.md` | How to actually see the app running. Compose Previews → install on phone → live edit. |
| 4 | `03-debugging.md` | Console / DevTools / DB inspector — Android equivalents. |
| 5 | `04-common-tasks.md` | "Change a string", "Change a button", "Add a screen" — translated from React. |
| 6 | `05-expo-and-rn-faq.md` | Why Expo Go won't run this app, and what the closest workflows are. |

Each file is short (≤ 200 lines). Skim, don't memorize.

> If you only have 10 minutes: read `00` and `02`.
