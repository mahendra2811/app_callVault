# Floating bubble during calls

The floating bubble is a tiny callNest overlay that appears while you are on a call. It shows the caller's tags, lead-score bucket, and last note — so you have context the moment the call connects.

## Permissions

The bubble needs the _Display over other apps_ permission. The first time you turn it on, Android sends you to a dedicated settings screen. Toggle callNest on and come back.

## Where it shows up

The bubble docks to the right edge of the screen, half-transparent. Tap to expand into a card with:

- Caller name + number
- Active tags
- Lead score (if enabled)
- Most recent note (truncated)

Drag to reposition; callNest remembers the spot for next time.

## Privacy

Nothing about the call leaves your phone. The bubble reads from the local database and disappears the moment the call ends.

## Disabling

**Settings → Real-time → Floating bubble**. You can also keep it on but show it only for unsaved numbers.

## Trouble

- _Bubble never appears_ → grant the overlay permission again.
- _Bubble appears blank_ → callNest is still resolving the number; usually clears within one second.
