# Filtering and search

callNest is built for finding the call you half-remember. Two tools do the heavy lifting: filters and full-text search.

## Filters

Tap the filter icon on the Calls tab. The bottom sheet groups options into Date, Direction, Tag, Lead score, and Saved status. Combine as many as you want — the active filter bar at the top of the list summarises them and lets you tap-to-remove individually.

Save a filter you use a lot via **Save as preset**. Presets appear in the overflow menu under _My filters_ and survive app restarts.

## Search

The magnifier in the top bar opens full-screen search. It looks across:

- Number
- Saved name
- Note text (every note, every revision)
- Tag name

Searching for `+91 98` matches any contact whose number starts with that prefix. Searching for `quote follow-up` looks inside notes.

Recent searches are remembered for 30 days. Clear them any time from **Settings → Privacy → Clear search history**.

## Tips

- Combine filter + search: filter to _unsaved + last 7 days_, then search inside that subset.
- Tap a result to jump to the call, or long-press for bulk actions.
